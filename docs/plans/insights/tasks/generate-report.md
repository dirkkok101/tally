### Acceptance Checks

- Generate validates unknown fields, versions, requested policies, rolling windows, category/metric/signal counts, horizon, and output bounds before TimeProvider or producer access.
- A valid request freezes TimeProvider.GetUtcNow exactly once before calling BudgetContractClient.GetInsightEvidenceAsync exactly once, binds the result, and compiles one complete InsightReport.
- Omitted selectors resolve to explicit released default window, formula, sufficiency, explanation, and ordering versions in the result.
- Success writes no ReportSnapshot, idempotency record, recovery metadata, file, or store state; expected failure returns one typed envelope and no partial report.
- CancellationToken is forwarded through every async call and logs contain only operation identity, correlation reference, duration, safe fingerprints, states, and counts.
