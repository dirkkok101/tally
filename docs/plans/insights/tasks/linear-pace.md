## Recipe

### Acceptance Checks

- For a started period with a non-zero plan, the exact baseline is plannedMinorUnits multiplied by elapsed inclusive calendar days divided by total period days with canonical exact rational arithmetic.
- Actual greater than the full plan returns OverPlan before baseline comparison; otherwise exact comparison returns BelowLinearBaseline, AtLinearBaseline, or AboveLinearBaseline.
- A zero plan with positive actual returns OverZeroPlan; zero or negative actual returns AtOrBelowZeroPlan without division.
- NoBudgetPlan, NoActiveBudgetPlanRevision, future period, or unavailable coherent evidence returns the exact named unavailable state with no baseline.
- Every result cites exact inputs, elapsed fraction, formula version, current-day incompleteness, uniform-distribution limitation, and explicitly says it is not a Forecast.
