### Acceptance Checks

- A compatible existing predecessor remains byte-semantically unchanged while current evidence is generated through the same coherent workflow and compared by source, policy, schema, exact metric, state, explanation, and attention sections.

- Do NOT validate only value changes; equal-value source, policy, and schema changes are explicit Restatements.
